# WorldOS 数据实时刷新技术方案（评审版）

## 一、需求分析

### 1.1 数据分类

| 类型 | 更新频率 | 示例 | 技术方案 |
|------|---------|------|---------|
| **高频数据** | 1-5分钟 | 大盘指数、VIX、油价 | 短轮询 |
| **中频数据** | 15分钟 | 资金流向、成交量 | 短轮询 |
| **低频数据** | 日级/周级 | GDP、CPI、M2 | 定时任务 |

### 1.2 评审结论（龙六）

- 秒级不可行：免费API不支持，需付费
- 5分钟足够：股票不需要秒级更新
- 现有架构可支持：无需额外后端

---

## 二、技术架构设计

### 2.1 简化架构（无需后端）

```
┌─────────────────────────────────────────────────────────────┐
│                        数据源层                               │
├─────────────┬─────────────┬──────────────────────────────────┤
│  高频API    │  中频API    │         低频文件                 │
│ (5分钟轮询) │ (15分钟轮询)│      (每日定时更新)              │
└──────┬──────┴──────┬──────┴───────────────┬─────────────────┘
       │             │                      │
       ▼             ▼                      ▼
┌─────────────────────────────────────────────────────────────┐
│                      前端展示层                              │
│              (统一轮询 + 缓存管理)                            │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 数据更新策略

| 数据类型 | 更新频率 | 技术方案 | 前端展示 |
|---------|---------|---------|---------|
| 大盘指数 | 5分钟 | 轮询 | 自动更新 |
| VIX/油价 | 5分钟 | 轮询 | 自动更新 |
| 资金流向 | 15分钟 | 轮询 | 自动更新 |
| M2/LPR | 日级 | 定时任务 | 静态显示 |
| GDP/CPI | 月级 | 定时任务 | 静态显示 |

---

### 2.3 评审结论

**不需要WebSocket/SSE**：
- 免费API不支持秒级更新
- 5分钟轮询足够满足需求
- 现有静态托管架构即可支持

## 三、具体实现方案

### 3.1 高频数据（WebSocket）

```typescript
// 前端 WebSocket 连接
const ws = new WebSocket('wss://api.example.com/ws');

// 接收高频数据
ws.onmessage = (event) => {
  const data = JSON.parse(event.data);
  updateHighFrequencyData(data);
};
```

**适用指标**：大盘指数、实时油价、VIX、美元指数

### 3.2 中频数据（短轮询）

```typescript
// 每5分钟轮询
const fetchMidFrequency = async () => {
  const response = await fetch('/api/mid-freq-data');
  const data = await response.json();
  updateMidFrequencyData(data);
};

setInterval(fetchMidFrequency, 300000); // 5分钟
```

**适用指标**：资金流向、成交量、北向资金

### 3.3 低频数据（定时任务）

```python
# Python 定时任务（每日16:30）
schedule.every().day.at("16:30").do(run_data_collection)
```

**适用指标**：GDP、CPI、M2、LPR

## 四、前端数据结构

```typescript
interface MarketData {
  // 高频数据（WebSocket推送）
  highFreq: {
    shIndex?: number;    // 上证指数

    oilPrice?: number;  // 油价

  // 中频数据（短轮询）
  midFreq: {
    northbound?: number;  // 北向资金

  };
  
  // 低频数据（定时更新）
  lowFreq: {
    gdp?: number;
    cpi?: number;
    m2?: number;
  };
  
  // 更新时间
  lastUpdate: {
    highFreq: string;
    midFreq: string;

  };
}
```
